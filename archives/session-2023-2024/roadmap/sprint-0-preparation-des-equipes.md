# Sprint 0 - Préparation des équipes

### Sprint 0

Note : le sprint 0 n'est pas prévu dans le Scrum Guide. Il s'agit d'un sprint "avant de pouvoir créer un incrément représentant une plus-value pour le client".

### Objectifs du sprint

* Former les équipes.&#x20;
* Configuration des environnements locaux de travail (sur leur propre machine).
* Prendre connaissance des [modalités d'évaluation](../../session-2022-2023/evaluation/) du projet.
* Lecture concernant le procès du "Libre" contre Microsoft.
